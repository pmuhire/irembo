## Installation

Install my-project with npm

```bash
  git clone https://github.com/pmuhire/irembo.git
  cd irembo
  npm install
  npx prisma 
```
### Build
```bash
  npm run build
```
### Start
```
  npm start
```
### Dev mode
 ```
  npm run dev 
  ```
